1. Data Storage

We store order and transaction data in Google Sheets for processing. Only essential fields are recorded, including your Order ID, User ID, provided device/contact info (e.g., "Android User"), service details, payment method, Transaction ID, and order status.

2. Realtime Status

Public status data is published to a Firebase Realtime Database to power the 'Live Status' page. This data includes non-sensitive information like Order ID, User ID, Service, and Status. We do not publish your full Transaction ID or other sensitive personal data in public feeds.

3. Notifications

Our administrators receive Email and Telegram notifications for new orders and exchanges to ensure fast processing. User-facing notifications are optional and subject to user consent (e.g., browser push notifications for order completion).

4. Third Parties

We do not use any third-party advertising trackers. We use Google Sheets and Google Apps Script for backend storage and processing, and Firebase for realtime database features. These services are governed by Google's Privacy Policy.

5. Contact

For any privacy-related inquiries, please contact us at support@easyrecharge.mm.
